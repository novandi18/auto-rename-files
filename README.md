## Installation
`npm i -g auto-rename-files`

## How to use
`autorename <path>`

## Arguments
| Name | Type | Description | Default |
|------|------|-------------|---------|
| Path | String | Your files path directory | './' |

## Examples
#### Linux
`autorename '/path/to/your/directory/'`
#### Windows
`autorename 'C:/path/to/your/directory/'`
